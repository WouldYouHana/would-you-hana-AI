# would-you-hana-AI
📁 우주하나 AI 레포지토리

- API 명세 : https://www.notion.so/API-11f6edd1e95780f28248c14e54e34288

## 사용법
- config 폴더 생성
- .env 생성 (API키 세팅) -> hana = OPEN AI APIkey
- 가상환경 생성
- pip install -r requirements.txt (라이브러리 설치)
- cd backend -> uvicorn main:app --reload (서버 실행)
